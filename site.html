O que são tags?

Tags são marcadores de conteúdo e são utilizadas para definir diferentes categorias de conteúdos a serem apresentados na página

Quais são as principais tags?

As principais tags existentes e disponívies em HTML5 podem ser divididas em N grupos distintos:

  1. Estrutura
  2. Semântica
  3. Textos e links
  4. Imagens
  5. Listas

Tags de Estrutura

<html></html>

A tag html atua como um delimitador dos elementos presentes na página, inclusive de apresentação e estrutura

Para que o '<' e o '>' não seja compreendido como parte do código, deve-se usar o '& lt' e '& gt' sem espaços e sem aspas

<head>

A tag head contem informações de metadados que serão lidos pelo navegador, como titulo, e a configuração de codificação

<body>

Responsável por conter todo o conteúdo visível da página

Tags Semânticas

<section>

É a mais genérica, podendo conter qualquer tipo de conteúdo

<header>

Corresponde ao cabeçalho da página

<article>

É utilizada para representar um conteúdo de relevância na página, e pode conter outra tags internas, como headers

<aside>

Geralmente é usado para representar conteúdo nas laterais da página

<footer>

É o rodapé da página/elemento em que esta contido. Pode ser usado para informaões de contato ou referencias

<h1> .. <h6>

Correspondem à hierarquia de titulos, sendo h1 a mais alta e h6 a mais baixa. Por boa prática, cada página deve possuir somente UM h1

Textos e Links

<p>

É utilizada para demarcar regiões com texto denso, incluindo imagens e links

<a>

Esta tag (anchor) é utilizada para demarcar hiperlinks, permitindo referenciar outros sites por meio do 'href=endereco.do site', e-mails por meio do href="mailto: email" e telefones href="tel:telefone"

Pode-se colocar um texto em substituição ao link, como nome para este, conforme mostrado abaixo

<a href="www.google.com">Google</a>

Outro importante elemento desta tag é o 'target' que dira ao navegador o que deve ser feito ao clicar no link, como por exemplo abrir em nova janela, por meio do parametro target="_blank"

<img>

Esta tag é utilizada para adicionar imagens à pagina, e possui apenas dois parametros, o 'src' que indica o diretório/link que contem a imagem e o 'alt' que é utilizado para representar um texto alternativo, caso o carregamento da imagem falhe

Listas

São utilizadas para dispor de conteúdo em tópicos ordenados ou não

  • <ul>: Lista não ordenada, com um marcador na frente do texto
  1. <ol>: Lista ordenada, com números ou letras frente do texto

O <li>: é um item de lista propriamente dito, pode ser utilizado sozinho, mas é interessante nos tipos acima descritos

O que um arquivo CSS?

Cascading Style Sheet (CSS) é um arquivo usado para estilizar elementos escritos em linguagem de marcação HTML. Uma das principais vantagens é que definido o estilo a ser utilizado, este pode ser usado para um diferente conjunto de paginas, desde que seja colocado em um arquivo externo e referenciado no head da página, entre as tags 'style'.

Box Model

O box model é a maneira pela qual o navegador apresenta os elementos de um html. Pode ser visualizado como uma caixa ao redor de cada elemento.

Os principais elementos do box model são explicitados abaixo:

  • Content: é o espaço que contém o conteudo propriamente dito, e possui fundo, uma cor de fonte ou uma imagem. Possui como dimensões a largura box de conteúdo, e altura do box de conteúdo.
  • Padding: é o espaçamento entre o conteúdo e a borda, sendo similar ao preenchimento de célula das planilhas eletronicas. Suas dimensões são a largura do padding-box e a altura do padding-box.
  • Border: é a margem em si, e possui como atributos a espessura (em px), o estilo (solid, dotted e dashed), e a cor (nome ou código hexa).
  • Margin: é o espaçamento entre a borda e os demais elementos do html. Suas dimensões são a largura e altura, podendo ser informadas em pares (dimensão vertical e horizontal), única (um tamanho para todos os lados) ou individual.

    • Estilizando texto

    Pode-se alterar também o texto dentro dos elemntos do html, há basicamente as seguintes propriedades que podem ser utilizadas

  • font-family: é a fonte propriamente dita. É recomendado sempre declarar ao menos duas, separadas por ',', pois caso uma delas não seja reconhecida pelo navegador, a outra será considerada
  • font-size: define o tamanho da fonte utilizada, sempre deve ser dimensionada em 'px'
  • font-style: define o estilo da fonte, 'normal' ou 'italic'
  • font-weight: define o tamanho da fonte, se esta é bold ou normal

    • Usando um CSS

    Uma maneira eficiente de usar o css é usar um arquivo externo, desta forma pode-se criar um estilo e fazer uso em diferentes páginas. Para isso, deve-se declarar o arquivo no 'head' da pagina, utilizando uma tag 'link'

    <link href="arquivos_de_estilo.css" rel="stylesheet">

    Definindo os estilos de cada tag

    Para definir qualquer estilo para qualquer tag, deve-se considerar a seguinte estrutura de regras:

    elemento_tag { property : value; }

  • elemento_tag: corresponde ao nome do elemento HTML a ser estilizados. Também é chamado de seletor
  • property: corresponde a propriedade do elemento HTML a ser estilizados, como a cor, o estilo e etc
  • value: corresponde ao valor da propriedade do elemento HTML a ser estilizados.

    • Tipos de seletores

    Há alguns tipos de seletores a serem usados e que devem ser explicitados

  • seletor de ID: é um elemento na página com o ID específicado. É referenciado como #um_id
  • seletor de classe: é um elemento na página com uma classe específicada. É referenciado como .uma_classe
  • hover: é usado para definir um estilo para um item quando o mouse é posicionado sobre este. É referenciado como seletor: hover

    • Usando Javascript

    Algo bem interessante é poder usar Javascript integrado à pagina html. Há duas formas distintas para esta integração, a primeira é a inclusão do código diretamente no documento HTML, sendo delimitados pela tag 'script'

    A segunda forma e mais recomendada, pois possibilita melhor separação de responsabilidades entre os elementos do projeto, bem como maior desempenho, é a utilização de um arquivo externo. Para usar esta forma, deve-se referenciar o arquivo externo dentro das tags 'script', como demonstrado abaixo

    <script src="js/meu_script.js"></script>
